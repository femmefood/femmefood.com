# Femme Food

This repository contains the source for [Femme Food](http://www.femmefood.com/)

## Dependencies

* [Jekyll](http://jekyllrb.com/)

## Run locally

```
jekyll serve -w
```

## Notes

* Post filenames may not contain æ, ø or å UTF-8 letters, since Disqus etc. got
  problems with UTF-8 letters.
