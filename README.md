# femme food

This repo contains the source for [femme food](http://www.femmefood.com/)

## Dependencies

* [Jekyll](http://jekyllrb.com/)

## Run locally

```
jekyll serve -w
```

## Notes

* Post filenames may contain æ and ø UTF-8 letters, but not å (for some reason it doesn't
  workout well on a Mac).
